# **App Name**: AcadBoost

## Core Features:

- User Authentication: Secure user login and registration with email verification.
- Personalized Dashboard: Display courses and track student's progress.
- Progress Tracking: Track student's progress through courses, including completed lessons, quiz scores, and time spent learning.
- Courses Offering: Offering courses from various resources using provided url/links.
- Quizzes: Give the user the ability to take quizzes
- Performance Analysis: Provide users with information of learning Gaps using their quizes result
- AI-Powered Recommendation Tool: Leverage AI to suggest personalized content and study materials based on student's learning patterns, and quiz history.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to represent trust and learning.
- Background color: Light gray (#F0F4F8) to provide a clean and modern feel...
- Accent color: A soft orange (#FFAB91) to highlight important actions and elements.
- Body font: 'PT Sans' for comfortable readability, ensuring content is accessible and clear. 
- Headline font: 'Playfair' (serif) paired with 'PT Sans' for body. Playfair adds elegance and readability in the headers, while PT Sans maintains readability in longer texts.
- Use flat, outline-style icons for a clean and modern look. Ensure icons are intuitive and easily recognizable.
- Subtle transitions and animations to provide feedback and enhance user experience without being distracting.
